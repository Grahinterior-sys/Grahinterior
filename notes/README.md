# Arch & Oak Website Project

## Overview
This is a professional furniture and architecture company website built with React and TailwindCSS.

## Project Structure
- `index.html`: Main entry with Tailwind theme configuration.
- `app.js`: Main application component.
- `components/`: Contains all UI sections (Header, Hero, Gallery, etc.).

## Maintenance Rules
- Check this README when adding new major sections.
- Ensure all images used are high-quality and professional to match the brand aesthetic.
- The theme uses CSS variables defined in `index.html` for easy color updates.
